# stroke_font

> 描边字

## 截止到flutter 1.17版本，暂无描边字功能。之前用shadow做出来的效果，不满意，还是需要绘制字体才是正宗的。

````dart
/// 参数

/// 文字内容
String fontStr

/// 文字大小
double fontSize, 
 
/// 可选参数一：描边颜色 default:Colors.black
Color strokeColor

/// 可选参数二：填充色 default:Colors.white
Color fillColor

/// 可选参数二：描边宽度 default: 2.0
double strokeWidth
````
